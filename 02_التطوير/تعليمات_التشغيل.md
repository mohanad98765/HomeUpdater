# 🚀 تعليمات تشغيل محدِّث المنزل (للتطوير)

> هذا الملف للمستخدم/المطوِّر الذي يريد تشغيل المشروع من المصدر.
> المستخدم النهائي سيستخدم البرنامج المُثبَّت من قائمة ابدأ بدلاً من ذلك.

---

## ⚡ التشغيل السريع (3 خطوات)

### 1️⃣ التثبيت لأوَّل مرَّة فقط
انقر مرتين على:
```
setup.bat
```
ستظهر نافذة UAC تطلب صلاحيات المسؤول — اقبلها. سيتولى السكربت تلقائياً:
- تثبيت Python 3.12
- تثبيت Node.js LTS
- تثبيت Nmap (لاكتشاف الأجهزة)
- إنشاء بيئة Python افتراضية
- تثبيت جميع الحِزَم المطلوبة

⏱️ المدَّة المتوقَّعة: **10–15 دقيقة** (حسب سرعة الإنترنت).

### 2️⃣ التشغيل اليومي
انقر مرتين على:
```
run.bat
```
سيقوم بالآتي تلقائياً:
- ✓ تشغيل الـ Backend على المنفذ 8000
- ✓ تشغيل الـ Frontend على المنفذ 5173
- ✓ فتح المتصفح على الواجهة

### 3️⃣ الإيقاف
أغلِق النافذتين:
- `HomeUpdater - Backend`
- `HomeUpdater - Frontend`

---

## 🔍 ماذا ستشاهد؟

عند فتح المتصفح، سترى الصفحة الرئيسية التي تعرض:

| العنصر | الوصف |
|--------|-------|
| 🟢 **متصل** | يعني أنَّ الـ Backend يعمل بشكل صحيح |
| 🔴 **غير متصل** | يعني أنَّ الـ Backend لم يبدأ — راجع نافذة Backend للأخطاء |
| **الإصدار** | رقم الإصدار الحالي للبرنامج |
| **حالة Backend** | الحالة المباشرة |

## 📜 ملفات السجلَّات (Logs)

كل عملية تشغيل تُحفَظ في مجلد `logs/` داخل `02_التطوير` تلقائياً، حتى تتمكَّن أنت أو Claude من مراجعتها لاحقاً.

| الملف | المحتوى |
|------|---------|
| `logs/setup_<timestamp>.log` | كل ما حصل أثناء تشغيل `setup.bat` |
| `logs/setup_latest.log` | نسخة من آخر setup (اقرأ هذا أوَّلاً عند المشكلة) |
| `logs/run_<timestamp>.log` | خطوات `run.bat` |
| `logs/run_latest.log` | نسخة من آخر run |
| `logs/backend_<timestamp>.log` | كل مخرجات FastAPI |
| `logs/backend_latest.log` | آخر تشغيل للـ Backend |
| `logs/frontend_<timestamp>.log` | كل مخرجات Vite |
| `logs/frontend_latest.log` | آخر تشغيل للـ Frontend |

> 💡 **عند ورود مشكلة**: أرسل لي محتوى `setup_latest.log` أو `run_latest.log` (حسب نوع المشكلة) وسأُشخِّص المشكلة بسرعة بدلاً من screenshots.

ملاحظة إضافية: الـ Backend يحفظ سجلَّاته الداخلية (أخطاء API، فحوصات الشبكة) في `%APPDATA%\HomeUpdater\logs\app.log`.

## 🐛 حلُّ المشاكل الشائعة

### ❌ "بيئة Python الافتراضية غير موجودة"
شغِّل `setup.bat` أولاً.

### ❌ "تعذَّر الاتصال بالخادم الخلفي"
1. تحقَّق من أنَّ نافذة **HomeUpdater - Backend** مفتوحة وتعرض رسالة `Uvicorn running on http://127.0.0.1:8000`.
2. إذا أُغلقت بسبب خطأ — راجع رسالة الخطأ في النافذة.
3. أعد تشغيل `run.bat`.

### ❌ المنفذ 8000 أو 5173 مستخدم
إذا كان لديك تطبيق آخر يستخدم نفس المنفذ:
1. أغلِق التطبيق الآخر، أو
2. عدِّل المنفذ في `backend/app/config.py` (للـ Backend) أو `frontend/vite.config.ts` (للـ Frontend).

### ❌ "Microsoft Defender SmartScreen prevented..."
انقر على **More info** ثم **Run anyway** — لأنَّ السكربت غير موقَّع رقمياً بعد (سنوقِّعه في مرحلة لاحقة).

### ❌ Nmap غير مُثبَّت
ربما رفض UAC تثبيته. أعد تشغيل `setup.bat` كمسؤول، أو ثبِّته يدوياً من https://nmap.org/download.

## 📁 أين يتمُّ حفظ البيانات؟

| النوع | المسار |
|-------|--------|
