# مشروع: تحديث كل شيء في المنزل
## Home Network Universal Updater

> برنامج موحّد لاكتشاف وتحديث كل أجهزة الشبكة المنزلية (كمبيوتر، جوالات، راوترات، تلفزيونات، IoT) من مكان واحد.

---

## كيف تستخدم هذا الدليل

هذا المجلد (`00_دليل_المشروع`) هو **مرجعك الرئيسي**. كل مرة ترجع للمشروع، ابدأ من هنا.

### ترتيب القراءة الموصى به:

| # | الملف | متى تقرأه |
|---|-------|-----------|
| 1 | **PROJECT_GUIDE.md** | في البداية، لتفهم الرؤية والنطاق |
| 2 | **PROGRESS.md** | عندما ترجع بعد فترة، لتعرف أين توقفت |
| 3 | **TASKS.md** | عند بدء جلسة عمل جديدة، لاختيار ما تعمل عليه |
| 4 | **ARCHITECTURE.md** | عند العمل على الكود أو اتخاذ قرار تقني |
| 5 | **DEVICES.md** | عند إضافة دعم لجهاز جديد |
| 6 | **DECISIONS.md** | عند الحاجة لفهم "لماذا" قررنا شيئًا |
| 7 | **BRAND.md** | عند تصميم أي عنصر بصري (شعار، واجهة، installer) |
| 8 | **WINDOWS_FUNDAMENTALS.md** | **مرجع إلزامي** قبل أي قرار معماري — checklist شاملة لكل مكوّن في برنامج Windows احترافي |
| 9 | **QUICKSTART.md** | عند الرغبة في تشغيل المشروع فعليًا |

## قاعدة ذهبية

> **في نهاية كل جلسة عمل، حدّث `PROGRESS.md` بما أنجزته وما هو التالي.**
>
> هذه القاعدة تضمن أنك تقدر ترجع للمشروع بعد أسبوع أو شهر، وتفهمه بالكامل خلال 5 دقائق.
